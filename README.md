# Tuan Docs

## Development

### Install Modules

```bash
pnpm install
```

If you have not installed `pnpm`, here is the official [installation guide](https://pnpm.io/installation).

### Update Modules

```bash
pnpm update
```

### Start Development Server

```bash
pnpm run docs:dev
```

### Build

```bash
pnpm run docs:build
```
